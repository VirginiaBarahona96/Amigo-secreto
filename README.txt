🎁 AMIGO SECRETO

Este es un proyecto simple para organizar un sorteo de Amigo Secreto. Permite agregar participantes, eliminar nombres, evitar duplicados y realizar el sorteo sin que nadie se asigne a sí mismo.

🚀 CARACTERÍSTICAS

✅ Agregar nombres de participantes.  
✅ Evitar nombres duplicados.  
✅ Eliminar participantes de la lista.  
✅ Sortear aleatoriamente sin autoasignación.  
✅ Mostrar los resultados del sorteo en pantalla.  
✅ Guardar los resultados en un archivo .txt.  

📂 ESTRUCTURA DEL PROYECTO

📂 amigo-secreto
│── 📄 index.html  # Página principal  
│── 📄 style.css   # Estilos del proyecto  
│── 📄 app.js      # Lógica del sorteo  
│── 📄 README.txt  # Documentación del proyecto  
│── 📂 assets      # Imágenes y recursos  

🛠️ CÓMO USAR

1. ABRIR `index.html` en un navegador.  
2. AGREGAR PARTICIPANTES escribiendo un nombre y presionando "Añadir".  
3. ELIMINAR UN PARTICIPANTE presionando ❌ junto a su nombre.  
4. REALIZAR EL SORTEO presionando "Sortear amigo".  
5. VER LOS RESULTADOS en la lista generada.  
6. GUARDAR LOS RESULTADOS en un archivo `.txt` presionando "Descargar resultados".
